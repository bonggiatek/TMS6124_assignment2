Project Reflection

This IoT project was a significant milestone in my development journey. Combining hardware and software components presented unique challenges that helped me grow as a full-stack developer. The system monitors environmental conditions and allows remote control of appliances through a secure web interface.

Key technologies used:

  • Raspberry Pi with Python for hardware interfacing
  • Node.js backend with Express
  • React.js frontend dashboard
  • MQTT for real-time communication
  • MongoDB for data storage
